# Food Order App

A full stack food ordering web application built using React.js, Node.js, and Express.js.

## Features
- Food menu display
- Add to cart
- Checkout system
- Order tracking
- REST API integration
- Responsive UI
- API testing using Jest and Supertest

## Tech Stack

Frontend:
- React.js
- Axios
- CSS

Backend:
- Node.js
- Express.js

Testing:
- Jest
- Supertest
